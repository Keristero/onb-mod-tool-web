# ONB Mod Tool

Browser-based WebAssembly analyzer for One Night Battle mod files.

## Usage

**Web Interface:**

Visit: https://keristero.github.io/mod-tool-keristerified/

**Local Development:**

```bash
cd src
mise run setup      # Install dependencies
mise run build:all  # Build everything
mise run serve:web  # Start local server
```
